Core Architecture Overview

The agdr_core crate implements the Atomic Kernel Inference (AKI) specification in safe Rust. It provides a zero-cost abstraction for cryptographically sealing AI decision records at the moment of inference.

Atomic Sealing BLAKE3 hashing and Ed25519 signing occur within a single, uninterruptible kernel boundary
PPP Triplet Binding Purpose, Place, and Person are cryptographically bound to each record at creation
Zero-Copy Design Minimizes memory allocations to achieve sub-microsecond latency
PyO3 Interoperability Seamless integration with Python via the agdr_python extension module

Atomic Kernel Inference Entry Point

use agdr_core::{AgDR, Purpose, Place, Person};

// Initialize with PPP Triplet
let record = AgDR::new(
    Purpose::from("High-risk credit evaluation"),
    Place::from("compute-node-alpha-east"),
    Person::from("FOI-7702")
);

// Execute inference with atomic sealing
let sealed = record.seal(|| {
    // Your inference logic here
    // Output is captured and hashed before release
    model.predict(input_features)
})?;

// Record is now immutable and court-admissible
assert_eq!(sealed.root_hash().len(), 32); // BLAKE3 output
assert!(sealed.signature().is_valid());   // Ed25519 verification

Integration Patterns

Embedding in a Rust Application

// Cargo.toml
[dependencies]
agdr_core = "0.0.2"

// src/main.rs
use agdr_core::{AgDR, Purpose, Place, Person};

fn main() -> Result<(), Box> {
    let record = AgDR::new(
        Purpose::from("Fraud detection inference"),
        Place::from("prod-cluster-us-west-2"),
        Person::from("system-account-9901")
    );
    
    let result = record.seal(|| {
        // Your model inference here
        fraud_model.evaluate(transaction_data)
    })?;
    
    // Persist or transmit the sealed record
    std::fs::write("record.bin", result.to_bytes())?;
    Ok(())
}

Using via PyO3 Python Bindings

# Install the Python extension
pip install agdr-python

# Use in Python
from agdr import AgDR, Purpose, Place, Person

record = AgDR(
    purpose=Purpose("Loan approval decision"),
    place=Place("inference-service-ca-central"),
    person=Person("compliance-officer-4410")
)

sealed = record.seal(lambda: credit_model.predict(applicant_data))
print(f"Record hash: {sealed.root_hash.hex()}")
